## 📖 Описание
Этот проект представляет собой персональный сайт-визитку, созданный для демонстрации навыков в разработке на **Python**, системном администрировании и веб-технологиях. Сайт включает в себя главную страницу «О себе», каталог проектов и детальную страницу конкретного кейса.

## 🛠 Технологии
В проекте использованы современные стандарты веб-разработки:
* **HTML5** — семантическая структура страниц.
* **CSS3** — использование **Flexbox** для построения сеток и **CSS Variables** для управления темами.
* **JavaScript (Vanilla)** — логика переключения тем и работа с **LocalStorage**.

## 🏗 Структура проекта
* `index.html` — главная страница с биографией и стеком технологий.
* `projects.html` — список реализованных проектов (Bank System, TG Shop Bot и др.).
* `details.html` — подробное описание технических особенностей проекта Bank Management System.
* `style.css` — стили, реализующие плавные переходы и адаптивность.
* `scripts.js` — скрипт для управления темой оформления.

## ✨ Особенности
1.  **Dark Mode:** Реализована полноценная темная тема. Выбор пользователя сохраняется при перезагрузке страницы благодаря использованию `localStorage`.
2.  **Адаптивная верстка:** Использование Flexbox позволяет сайту корректно отображаться на мобильных устройствах и десктопах.
3.  **Интерактивность:** Карточки проектов имеют эффект при наведении (`hover`) с приподниманием и тенью.
4.  **Чистый код:** Проект разделен на логические модули, что упрощает поддержку и расширение функционала.
